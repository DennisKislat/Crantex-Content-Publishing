🔄 Integration Flow

This demo shows the complete end-to-end integration with the TikTok Content Posting API.

OAuth 2.0 Authorization

User clicks "Login with TikTok" → redirected to TikTok OAuth → returns access token

Query Creator Info

App calls /v2/post/publish/creator_info/query/ to verify the user and get posting capabilities

Initialize Video Upload

App calls /v2/post/publish/video/init/ with video metadata to get upload URL

Upload Video Chunks

App uploads the video file to TikTok's upload endpoint

Content Published

Video appears on the user's TikTok account with the specified privacy level
